# ⏳ Pomodoro Timer

Um timer Pomodoro desenvolvido com HTML, CSS e JavaScript para ajudar na produtividade e foco durante os estudos ou trabalho.

## 📌 Sobre o projeto

Este projeto foi criado com o objetivo de praticar manipulação de DOM, controle de tempo com JavaScript e organização de layout com CSS.

O método Pomodoro consiste em:
- 25 minutos de foco
- 5 minutos de pausa curta
- Pausa longa após 4 ciclos

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🎯 Funcionalidades

- Iniciar, pausar e resetar o timer
- Alternar entre foco e pausa
- Contagem regressiva dinâmica
- Alerta sonoro ao finalizar o tempo
- Interface responsiva

## ▶️ Como executar o projeto

1. Clone o repositório: https://github.com/OiNico/Pomodoro-Timer.git

2. Abra o arquivo `index.html` no navegador.

Ou simplesmente abra o projeto clicando duas vezes no arquivo HTML.
